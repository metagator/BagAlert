BagAlert for WotLK 3.3.5a.

BagAlert watches your total free bag slots and pops an alert when you hit (or drop below) your chosen threshold. You can run it icon + text, text-only, sound-only, or any combo that fits your UI. The frame is draggable (and lockable), and everything is configurable through the Blizzard Interface Options panel.

Commands

/ba or /bagalert — open settings

/ba test — preview the alert

/ba sound — play the sound

/ba reset — reset position
